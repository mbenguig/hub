Getting started with packages
Want to create a new package for ProActive? Here are a few steps to get started:
  1. Read the Create and Contribute documentation page.
  2. Take a look at the sample package and some examples.
  3. Use our package template on GitHub as a structure reference.
Submit a package

All contributions to ActiveEon Hub are welcome! If your pack might be useful to more people than just yourself, we would be happy to include it.

Don't worry if your code isn't perfect or is incomplete. We will organize ourselves to come and help.
